# Contributing

Thank you for your interest in contributing to the Python Data Science Tutorial!

## How to Contribute

1. **Fork the Repository**: Click the "Fork" button on GitHub.

2. **Clone Your Fork**: `git clone https://github.com/yourusername/python-data-science-tutorial.git`

3. **Create a Branch**: `git checkout -b feature/your-feature`

4. **Make Changes**: Implement your feature or fix.

5. **Test Your Changes**: Run tests with `python -m pytest tests/`

6. **Commit Changes**: `git commit -m "Add your message"`

7. **Push to Your Fork**: `git push origin feature/your-feature`

8. **Submit a Pull Request**: Open a PR on GitHub.

## Guidelines

- Follow PEP 8 style guidelines.
- Add unit tests for new features.
- Update documentation as needed.
- Ensure all tests pass.
- Keep commits focused and descriptive.

## Reporting Issues

Use GitHub Issues to report bugs or suggest features.

## Code of Conduct

Be respectful and inclusive.
